Exemplo de como informar o caminho de entrada e saida:


"cd "C:\Users\fabia\OneDrive\Desktop\Trabalho arquitetura\LeitorAssembly\src":" Altera o diretório atual para onde estão localizados os arquivos fonte Java.
 É necessário estar no diretório correto para compilar e executar os arquivos.


"javac LeitorAssembly.java ConversorDeInstrucoes.java ManipuladorArquivos.java Listas.java:"
Compila os arquivos Java. O comando javac transforma os arquivos .java em arquivos .class, que são necessários para a execução do programa.


"java LeitorAssembly "C:\Users\fabia\Downloads\programa3.txt" "C:\Users\fabia\OneDrive\Desktop\Saida\Versao_Teste5.txt":"
Executa o programa Java principal (LeitorAssembly) e fornece os caminhos para os arquivos de entrada e saída. 
O primeiro argumento é o caminho para o arquivo de entrada e o segundo é o caminho para o arquivo onde a saída será gravada.
